Чем ты занимаешься?

Я делаю много вещей. В настоящее время я изучаю Docker и улучшаю свои навыки написания сценариев bash чтобы стать DevOps инженером. У меня есть полностью рабочий самодельный домашний NAS (Debian 11, 1ТБ) из старого компьютера. У меня есть автономный bittorrent-клиент (Deluge) для моих ISO-образов Linux, менеджер контейнеров (Yacht), медиасервер (Jellyfin) и облачное хранилище (Nextcloud), при этом все они работают в контейнерах.
У меня также есть несколько других проектов на моей Github странице, большинство из них — это игровые площадки для изучения Python.
Я использую Arch кстати.
